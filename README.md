# Hüseyin Karacif | Kişisel Portfolyo Sitesi

Kişisel portfolyom için oluşturduğum modern, responsive ve animasyonlu web sitesi.

## Özellikler

- **Modern Tasarım**: TailwindCSS ile geliştirilmiş karanlık mod odaklı modern arayüz
- **Responsive (Duyarlı) Düzen**: Tüm cihazlarda (mobil, tablet, masaüstü) sorunsuz görünüm
- **Eğlenceli Mini Oyunlar**:
  - Yılan Oyunu (Snake)
  - Tic-Tac-Toe (Yapay zeka rakibe karşı)
  - Hafıza Oyunu (Memory)
  - Ping Pong
  - Mayın Tarlası
- **Animasyonlar**:
  - Kaydırma sırasında yumuşak görünüm (scroll reveal)
  - Özel fare imleci (custom cursor)
  - CSS keyframe arka plan animasyonları

## Teknolojiler

- HTML5
- CSS3 (TailwindCSS)
- Vanilla JavaScript
- Font Awesome

## Kurulum ve Çalıştırma

Projeyi yerel ortamınızda çalıştırmak için herhangi bir ek kuruluma ihtiyaç yoktur:

1. Depoyu bilgisayarınıza klonlayın:
   ```bash
   git clone https://github.com/huseyinkaracif/whoiskaracif.git
   ```
2. Proje dizinine gidin.
3. `index.html` dosyasını favori web tarayıcınızda açın.

Alternatif olarak, VS Code eklentisi olan "Live Server" ile de projeyi hızlıca çalıştırabilirsiniz.
